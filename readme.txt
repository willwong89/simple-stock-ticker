=== Plugin Name ===
Contributor: William Wong
Tested up to: 4.2.2

This plugin creates a shortcode to output the current price for a single stock ticker symbol. It relies on the Yahoo! Finance API.

== Installation ==

1. Upload 'simple-stock-ticker.php' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [stock-ticker s="SYMBOL"] in your post / page

== Screenshots ==

1. Shortcode with default parameters (name = "on", symbol ="on", change="on", color="on", price="on") and the output of it. 
2. Shortcode with parameters (name = "off", price="off") and the output of it.
3. Shortcode with parameters (symbol = "false", change="false", color="false") and the output of it.
